# tdc v0.1.2

This maintenance release removes macOS metadata files from the repository and release source tree. It keeps the user-owned installation and password-free updater introduced in v0.1.1, with `tdc` and `tdc-drive9` managed together under `~/.tdc/bin`.

Upgrade an existing v0.1.1 installation with:

```bash
tdc update --check
tdc update
tdc --version
```
